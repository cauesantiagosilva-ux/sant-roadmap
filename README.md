# Sant Roadmap

Um guia completo para se tornar um desenvolvedor full stack, estruturado em fases progressivas desde os fundamentos até desenvolvimento avançado.

## Características

- Trilha de aprendizado estruturada
- Conteúdo em português
- Exercícios práticos
- Interface moderna e responsiva
- Recursos gratuitos selecionados

## Como Usar

1. Acesse o site em [sant-roadmap.github.io](https://sant-roadmap.github.io)
2. Escolha sua trilha: Fundamentos, Front-end ou Back-end
3. Siga o conteúdo sequencialmente
4. Complete os exercícios práticos
5. Acompanhe seu progresso

## Desenvolvimento Local

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/sant-roadmap.git

# Entre na pasta do projeto
cd sant-roadmap

# Abra o index.html no seu navegador
```
